PDF Summarizer with Amazon Bedrock & Claude 3.7
A generative AI application that summarizes and answers questions about PDF documents using Amazon Bedrock, Claude 3.7 Sonnet, and Streamlit.

This project demonstrates how to use the Bedrock Converse API to send PDF document bytes directly to the model, eliminating the need for external OCR or text extraction libraries.

Direct PDF Processing: Sends raw PDF bytes to Claude 3.7.

Interactive UI: Simple interface built with Streamlit.

No Vector DB Required: Uses the model's long context window for immediate document analysis.

‚ö†Ô∏è Troubleshooting
Error: AccessDeniedException

Fix: Ensure your IAM user/role has the bedrock:InvokeModel permission and that you have enabled Claude 3.7 Sonnet in the Bedrock Model Access page in the specific region you are using (e.g., us-east-1 or us-west-2).

Error: ResourceNotFoundException

Fix: Check that the modelId in summarization_lib.py matches the ID available in your region. The code currently uses us.anthropic.claude-3-7-sonnet-20250219-v1:0.
